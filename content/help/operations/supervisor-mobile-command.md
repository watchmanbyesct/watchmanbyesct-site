Supervisors can manage **field and desk operations** from a dedicated **mobile layout** (`/supervisor-mobile`).

### What is available

- **Field mode:** dashboard, schedule, timekeeping, DAR, patrol, incidents, maintenance, handover
- **Desk mode:** command dashboard, guard status, incidents, dispatch, scheduling, DAR review
- **Support modules:** sync center, reports, communications/messages, checkpoint programming, post orders
- **Desktop module access from mobile:** command center, dispatch/incidents desks, scheduling workspace, intelligence, live map, audit/admin pages (when entitled)

### When to use it

Use it when you are away from the desktop but still need command visibility and action capability, especially for incident and dispatch follow-up.

### Access notes

What appears depends on role plus module gating. If an expected menu item is missing, confirm both your role permissions and your tenant module configuration.
