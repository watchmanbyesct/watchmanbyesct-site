Supervisors can manage **field operations** from a dedicated mobile shell: guard status, incidents, dispatch, inspections (where enabled), DAR review, and sync status.

### When to use it

Use it when you are away from the desktop but need command visibility—especially incident and dispatch follow-up.

### For operators

The shell includes online/offline indicators and a drawer that links mobile routes back to desktop modules where appropriate.
